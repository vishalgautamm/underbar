# Underbar.js

## Underbar (an implementation of Underscore) is a JavaScript library that provides useful functional programming helpers.

The following methods have been implemented:

### Collections
- identity
- first
- last
- each
- indexOf
- filter
- reject
- uniq
- map
- pluck
- reduce
- contains
- every
- some

### Objects
- extend
- defaults

### Functions
- once
- memoize
- delay
- shuffle
- invoke
- sortBy
- zip
- flatten
- intersection
- difference
- throttle
